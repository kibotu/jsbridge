Quick start

await jsbridge.ready();
jsbridge.setDebug(true);

jsbridge.on((msg) => {
  const { action, content } = msg.data || {};
  console.log(action, content);
});

const info = await jsbridge.call({ data: { action: 'deviceInfo' } });

API

  • jsbridge.ready(): Promise<void>
  • jsbridge.call(msg, opts?): Promise<any>
  • jsbridge.on(handler) / jsbridge.off(handler)
  • jsbridge.setDebug(bool)
  • jsbridge.setMockHandler(fn | null)
  • jsbridge.platform — 'android' | 'ios' | 'desktop'
  • jsbridge.schemaVersion — integer

Message format

{
  version: jsbridge.schemaVersion,
  id: "msg_...",
  data: { action: "...", content: { ... } }
}

CSS Custom Properties

Native automatically pushes --bridge-inset-* CSS properties whenever bars change. Use var(--bridge-inset-top) etc. in your CSS. The getInsets action is for on-demand JavaScript queries.

Spec & Usage 
/* CSS: automatically updated by native */
body {
  padding-top: var(--bridge-inset-top, env(safe-area-inset-top, 0px));
  padding-bottom: var(--bridge-inset-bottom, env(safe-area-inset-bottom, 0px));
}

/* JS: query on demand */
const insets = await jsbridge.call({ data: { action: 'getInsets' } });
// { statusBar: { height, visible }, systemNavigation: { ... }, safeArea: { top, right, bottom, left } }
